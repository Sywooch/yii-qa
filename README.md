Yii-QA
======

感谢选择 Yii-QA, 基于[Yii2](https://github.com/yiisoft/yii2)框架基础实现的问答程序.


环境条件
------------
- >= php5.4
- Yii2 


安装
------------
本程序基于命令行安装:
- 编辑 ``protected/config/db.php'文件, 填入您的数据库配置, 请确认数据库存在.
- 打开命令行工具 -> 切换到程序根目录 -> 执行 ``php yii migrate`` 执行数据迁移命令导入数据结构
